AMR-GraphNet : AMR Engraftment Network for FMT

Suhashi Nihara De Silva, Sakesh Andhavarapu, Alejandro Dale Kimball, Christina Boucher, Ilya B Slizovskiy

Antimicrobial resistance (AMR) transfer during fecal microbiota transplantation (FMT) is an important consideration for both safety and therapeutic effectiveness. Our AMR-FMT analysis investigates how resistance genes are distributed and colocalized before and after FMT. The study is based on curated datasets that capture antimicrobial resistance (AMR) genes and mobile genetic element (MGE) associations across donors, pre-FMT, and multiple post-FMT timepoints and includes data from three patient groups: Melanoma, multidrug-resistant bacteria (MDRB), and recurrent Clostridioides difficile infection (rCDI) along with matched donor samples. The analysis compares donor, pre-FMT, and post-FMT samples, highlighting how AMR gene abundance, diversity, and gene colocalization patterns shift across conditions and over time. This structured approach allows systematic comparison of AMR dynamics across diseases and patients, highlighting how donor selection and patient background can influence the persistence or loss of resistance genes after FMT.

Data & Input

Patient data was collected from multiple studies covering three different disease groups Melanoma, multidrug-resistant bacteria (MDRB), and recurrent Clostridioides difficile infection (rCDI). These datasets provide paired donor, pre-FMT, and post-FMT samples, enabling tracking of antimicrobial resistance (AMR) dynamics. The input to the AMR-FMT tool is provided as a CSV file with the following structure:

CSV structure


          Patient ID, Disease type, AMR ID, MGE ID, Donor, PreFMT, PostFMT(timepoints)
Header Description
Patient ID Unique patient identifier
Disease type Type of disease (e.g., Melanoma, MDRB, rCDI)
AMR ID Classification of AMR genes
MGE ID Specific mobile genetic element associated with AMR
Donor 1 if the colocalization (gene and MGE combination in the row) is present in the donor sample, 0 if absent
PreFMT 1 if the colocalization is present in the pre-FMT sample, 0 if absent
PostFMT (timepoints) 1 if the colocalization is present in post-FMT samples at various timepoints, 0 if absent

Output

We provide a comprehensive output of the analysis, including both interactive visualizations and data summaries. The output highlights the frequency of antimicrobial resistance gene (ARG) and mobile genetic element (MGE) combinations across different phases of FMT. Specifically, the results include the number of patients in which each ARG–MGE colocalization is observed in the donor, pre-FMT, and post-FMT stages.
